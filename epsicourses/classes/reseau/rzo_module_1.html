Les protocoles
NAT / PAT

Des acronymes expliqués :

  • NAT : Network Address Translation
  • PAT : Port Address Translation

Le protocole NAT Statique est bien moins intéressant que le NAT Dynamique, car il est moins performant (point de vue sécurité, une porte est ouverte et tout le monde est invité).

Il existe divers types de NAT dynamique :

  • Le NAT à cône plein
  • Le NAT à cône restrictif
  • Le NAT à cône restrictif sur les ports
  • Le NAT symétrique
